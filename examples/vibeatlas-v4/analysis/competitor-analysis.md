# Competitor Analysis - AI Coding Assistants Market

## Market Overview

**Market Size:** 10M+ developers using AI coding tools (2025)
**Growth Rate:** 300% YoY
**Key Players:** GitHub Copilot (dominant), Cursor (fast-growing), Tabnine, Cody, etc.

---

## GitHub Copilot

**Market Share:** ~60%
**Pricing:** $10/month (individual), $19/user/month (business)
**Users:** 6M+ (estimated)

**Strengths:**
- Default choice (bundled with GitHub)
- Large training dataset
- Simple to use
- Works across editors

**Weaknesses:**
- Black box (zero transparency)
- No context control
- No cost visibility
- Can't optimize usage
- Suggestions often irrelevant

**User Complaints (from Reddit/Twitter):**
- "Copilot suggests code from wrong files"
- "No way to see what context it's using"
- "Wish I could control what it sees"
- "Sometimes completely wrong suggestions"
- "No way to optimize for quality"

---

## Cursor

**Market Share:** ~15%
**Pricing:** $20/month (pro), $40/month (heavy usage)
**Users:** 1.5M+ (estimated)

**Strengths:**
- Better AI than Copilot
- Some context control
- Good UX/design
- Fast responses

**Weaknesses:**
- Expensive (2-4x Copilot)
- Costs can spike unexpectedly
- Still limited transparency
- No preventive controls
- VS Code fork (not extension)

**User Complaints:**
- "Love it but too expensive"
- "Bill went from $20 to $40 unexpectedly"
- "Wish I could see token usage"
- "Hard to optimize costs"
- "No way to cap spending"
